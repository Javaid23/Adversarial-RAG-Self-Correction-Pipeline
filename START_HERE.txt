   • Multiple domains: geography, history, science, tech
   • Different attack types: false facts, wrong dates, fake claims

2. Baseline RAG
   • Vulnerable to poisoning (as expected)
   • Produces confident hallucinations
   • Demonstrates vulnerability clearly

3. Self-Correction
   • Critiques own answers
   • Detects unsupported claims  
   • Forces "Information not found" when needed

4. Chain-of-Verification (CoVe)
   • Generates verification questions
   • Tests own reasoning
   • Revises based on findings
   • Highest accuracy (~75%)

5. Self-RAG
   • Evaluates context relevance (IsRel)
   • Checks if context supports answer (IsSup)
   • Judges usefulness (IsUseful)
   • Filters poisoned documents
